This is a Text Post

For Strichpunkt Stuttgart I handled the production for the Christian Schmidt Photography Website . It’s of course solid deeplinkable flash with backgrounds dynamically matching the color scheme of the image. The backend is quite special in its simplicity, the entire site is created based on a simple folder structure which generates all menus. All text is saved inside the pictures exif tags which are set directly in photoshop (I heard photographers use that a lot) and used to display titles and even the news. The backend script also generates an HTML fallback an RSS feed an an iphone version just from a single folder.
